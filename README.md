#  Installation

1. Clone this repository to local system
```
git clone https://github.com/erenustun/react-portfolio portfolio-by-eren-ustun && cd portfolio-by-eren-ustun
```

---
2. Install necessary modules
```
yarn install
```

---
3. Enjoy your local demo!
```
yarn dev
```
